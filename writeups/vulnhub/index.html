VulnHub - Alfa

Alfa starts with enumeration on FTP to obtain a username and an image file, and the image file is named after the user’s pet. Alfa’s website has a robots.txt file which contains an obfuscated web path to intranet chat support. The chat conversation reveals sensitive information, and with this, I’m able to guess the user’s password and gain a foothold on the system. In the user’s home directory, there is a VNC password and it can be used to log into the currently running VNC server as root. ...

VulnHub - DC-9

DC-9 from VulnHub features a website that is vulnerable to SQL injection. I’m able to dump a bunch of users’ credentials by leveraging the SQLi and gain a foothold a on the system after spraying these credentials on SSH. One of the users has a sudo privileges on a custom binary which allows me to perform an arbitrary file write with root access. Actually, there is a port knocking rule in this machine to open the SSH port, but when I first solved this machine, my full nmap scan broke that rule. ...

VulnHub - DC-6

DC-6 starts off by enumerating usernames from a WordPress website. I’m able to gain a set of credentials to log into the admin panel with a brute-force attack. There is a WP Plugin which can be leveraged to gain a foothold on the system. A to-do note containing the user’s credentials is discovered while enumerating the home directory. For the root part, there is a sudo privileges on a writable backup script and nmap which allows me to escalate to other users then straight to root. ...
